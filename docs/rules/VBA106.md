# VBA106 вЂ” Erase target is not an array

**Severity:** рџ”ґ error    **Category:** `declaration`    **Phase:** 1.4

## Description

`Erase` requires an array. Scalar variables cannot be erased.

## Failing example

```vb
Dim s As String
Erase s
```

## Compliant example

```vb
Dim arr() As Long
Erase arr
```

## How to fix

Use `s = vbNullString` for a String, or declare an array.
