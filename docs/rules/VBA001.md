# VBA001 — Undefined identifier

**Severity:** 🔴 error    **Category:** `name_resolution`    **Phase:** 0

## Description

A symbol is referenced that is not declared in any visible scope.

## Failing example

```vb
Sub S()
    typo = 1
End Sub
```

## Compliant example

```vb
Sub S()
    Dim count As Long
    count = 1
End Sub
```

## How to fix

Declare the variable with `Dim` or fix the spelling.
