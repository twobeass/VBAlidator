# VBA003 — Duplicate declaration

**Severity:** 🔴 error    **Category:** `declaration`    **Phase:** 0

## Description

Two `Dim`/`Const` declarations define the same name in the same scope.

## Failing example

```vb
Dim x As Long
Dim x As String
```

## Compliant example

```vb
Dim x As Long
Dim y As String
```

## How to fix

Rename one of them or remove the duplicate.
