# VBA231 вАФ `Const` initialiser references a non-constant

**Severity:** рЯФі error    **Category:** `const_expression`    **Phase:** 2.5

## Description

`Const` initialisers cannot reference variables вАФ only literals, other constants, or enum members.

## Failing example

```vb
Dim runtimeValue As Long
Const X As Long = runtimeValue
```

## Compliant example

```vb
Const A As Long = 5
Const X As Long = A + 1
```

## How to fix

Use a literal or another `Const` / Enum member as the initialiser.
